# Bid Service Plan

## Objective
Build a low-cost, service-first public bid monitoring business where Zee handles most of the execution and Dylan approves meaningful external actions when needed.

## Initial Niche
- Janitorial / custodial / facility cleaning for public buildings and schools.

## Core Offer
- Monitor public bid sources daily for matched janitorial opportunities.
- Filter and summarize opportunities by geography and relevance.
- Deliver deadlines, scope highlights, pre-bid meeting dates, and direct links to bid docs.

## Why This Niche
- Frequent rebids
- Clear language
- Large small-business buyer pool
- Good fit for AI-assisted monitoring and summarization
- Low startup cost

## Initial Ideal Customer Profile
- Owner-led janitorial company
- 10–50 employees
- Already does commercial cleaning
- Wants larger recurring contracts
- No full-time bid coordinator
- Operates in one metro or one state

## First Service Tiers
### Tier 1 — Local Alerts
- One metro / county region
- Daily matched bid alerts
- Basic summaries
- Direct links to source docs

### Tier 2 — Qualified Opportunities
- Broader region / statewide
- Everything in Tier 1
- AI-cleaned scope summaries
- Priority ranking
- Weekly digest

### Tier 3 — Bid Desk Assist
- Everything in Tier 2
- Attachment triage
- Key compliance flags
- Walk-through date tracking
- Quick fit scoring

## First Execution Stages
1. Build workspace structure
2. Build janitorial source list
3. Create opportunity brief template
4. Draft outreach / offer copy
5. Build first prospect list
6. Produce first sample briefs
7. Prepare for first outreach batch

## Constraints
- Prefer free / low-cost methods
- No money spent unless Dylan explicitly authorizes it
- External sends should remain deliberate and controlled
- Security and data hygiene remain high priority
