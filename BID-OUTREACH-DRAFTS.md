# Bid Outreach Drafts

## Anti-Spam Operating Rule
- Start with a small selective batch only.
- Use only targets that clearly fit commercial/public-facing janitorial work.
- Personalize lightly but truthfully.
- Stop after a small first batch and assess signal before expanding.

## Initial Batch Size
- Recommended first batch: 5 companies max
- Do not send more until quality review and response review happen
