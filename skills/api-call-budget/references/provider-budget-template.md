# Provider Budget Template

Use this as a quick operating note for a sensitive provider.

```markdown
# Provider Budget

- Provider:
- Task:
- Date started:
- Restriction history:
- What counts as one call:
- Estimated safe session budget:
- Soft stop:
- Hard stop:
- Batch size:
- Cooldown between batches:
- Fallback path:
- Current state: active | paused | cooldown | blocked
- Notes:
```

## Suggested usage

- Fill this in before a risky workflow starts.
- Update after each meaningful batch.
- Keep one note per provider or per project-provider pair.
