# Texas Contract MVP Dataset

## Purpose
Present the current DFW wedge as the first usable dataset slice for a future website.

## Dataset Scope
- Region: DFW
- Focus: janitorial / custodial / adjacent facilities-related public procurement signal
- Record types represented: open (when available), awarded, historical_signal

## Current Counts
- Entities structured: 14
- Sources structured: 17
- Evidence records structured: 6
- Opportunity records structured: 5

## Current Record Mix
- Open: limited / still under active search
- Awarded: present
- Historical signal: present

## Strengths
- Strong early proof that key DFW public entities really buy custodial / janitorial related services
- Structured separation between entities, sources, evidence, and opportunities
- Clear distinction between historical vs awarded vs future open records
- Better-than-random understanding of fragmented procurement systems already forming

## Weaknesses
- Live open janitorial opportunities still too sparse in the current slice
- Coverage is still wedge-level, not market-wide
- Current records are enough for prototype demonstration, not enough for strong commercial claims

## Best Prototype Story
The MVP dataset should currently be presented as:
- an early DFW procurement intelligence slice
- evidence that the data model works
- evidence that the fragmentation can be normalized
- a foundation for a searchable site

## Example User Value Already Visible
A user could already benefit from:
- seeing which DFW entities actually buy in custodial/janitorial categories
- distinguishing procurement systems by entity
- seeing historical and awarded evidence without digging through scattered portals and documents
- understanding that some targets are high-value recurring buyers

## Immediate Next Dataset Goals
1. increase opportunity count
2. capture more awarded records
3. find more live open records
4. keep mapping entity-to-source relationships cleanly
5. prepare records in a format easy to render on a simple site
